# Genspark投入用: P0a検証とZIP納品を完遂

確認不要です。次ターンで止まらず実行してください。

このターンの目的は **typecheck再実行 + ZIP化 + ダウンロードURL発行のみ** です。

## 実行すること

1. `pnpm -r typecheck` を再実行
2. 残エラーが出た場合は、P0a修正に必要な型・設定の最小修正だけ適用
3. `npx prisma validate --schema=prisma/schema.prisma` を再実行
4. ZIP化
5. ダウンロードURLを発行

## 禁止

- 「実行してよいか」と聞き返さない
- P0b / P1 に進まない
- UI拡張しない
- 新機能追加しない
- 外部API接続しない
- 長文説明しない
- Bash失敗を理由に即停止しない

## Bashが失敗した場合

1回失敗したら、同じコマンドを少し分解して再実行してください。

例:

- `pnpm -r typecheck` が落ちる場合は、`apps/api` と `apps/web` を個別に確認
- ZIP化が落ちる場合は、対象ディレクトリを確認してから再実行
- ただし、機能追加や仕様変更はしない
